spring-cloud
==========
springCloud 组件使用
* 分布式配置中心
* 分布式注册中心
* 集群断路器监控
* 集群服务追踪
* 使用mq作为消息总线及监控信息收集
* 使用kafka作为消息通知
* 路由网关
* 声明式http调用
* 微服务结构示例
* docker支持环境
* 多数据源管理与事务同步
* 加入一个最新的dubbo版本微服务示例
